# 🔑 pwgo
`pwgo` is a fast, modern and optimized CLI Tool.

## 🤔 Why pwgo?
* **Secure**: Uses `crypto/rand`, fully secure numbers.
* **Fast**: Built with `strings.Builder`.
* **Customizable**: Length, quantity, and character pool.
